# pintos-user-programs
Implementing user programs in PintOS

## Prerequisites 
QEMU emulator

## Installation

1. `cd ~`
2. `git clone https://github.com/suj97/pintos-user-programs.git`
3. `cd pintos-user-programs/pintos/src/utils`
4. `make`
5. `cd ../threads`
6. `make`
7. `./pintos run alarm-multiple`
